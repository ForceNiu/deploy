# 山水行旅 -- 武夷山 · 庐山 · 九江 9天旅行计划

> 竹筏 · 茶香 · 云海 · 慢生活 | 2026.6.12 - 6.20

## 项目简介

一份精心规划的旅行计划 Web 应用，涵盖武夷山（4天）、庐山（2天）、九江（2天）、转场（1天）共9天的完整行程。纯静态页面，无需后端，部署在 Cloudflare Pages 上，移动端优先设计，支持深色模式。

**在线访问**: https://travel-plan-3cu.pages.dev

## 功能特性

### 每日行程

- 9天完整行程（Day12-Day20），每天按时间块（L0 概览 / L1 摘要 / L2 详情）三层展开
- 每个时间块包含：出发时间、游玩时长、详细描述、交通信息、注意事项
- 真实游客评价（来源：携程、猫途鹰、去哪儿、Trip.com）
- 雨天备选方案，天气数据联动高亮
- 次日预告，提前规划
- 高德地图路线规划按钮（自动收集当天景点坐标生成导航链接）
- 住宿信息展示（酒店名称、地址、评分、导航链接）
- 今日 CheckList 提醒（localStorage 持久化勾选状态）

### 天气预报

- 调用 Open-Meteo API 获取武夷山、庐山、九江三城市实时天气
- 同时获取去年同期历史天气作为参考
- 城市筛选 Tab（全部 / 武夷山 / 庐山 / 九江）
- 温度可视化色条（蓝到红渐变）
- 降水概率显示
- 高温 / 暴雨天气预警
- 日出时间动态更新，自动计算出发时间
- 穿搭建议（基于温度、降水、行程强度自动生成）
- 骨架屏加载态 + 手动刷新

### 美食推荐

- 28道地方特色美食（武夷山 11 道、庐山 6 道、九江 11 道）
- 按城市筛选 + 按类型筛选（早餐 / 正餐 / 小吃 / 甜品）
- 评分、价格区间、餐厅位置、高德导航链接
- 每日行程卡片内嵌当日美食推荐（横向滚动卡片）
- 景点与美食双向关联（景点详情中可跳转相关美食）

### 伴手礼推荐

- 9款地方特产（武夷山 3 款、庐山 3 款、九江 3 款）
- 按城市筛选，显示标签（必买 / 茶 / 非遗 等）
- 价格参考、购买地点

### 门票价格汇总

- 9项门票/交通费用明细表
- 分类合计（景区门票+交通 / 景点门票）
- 购票渠道说明（小程序 / 现场购票）
- 购票提醒（提前抢票时间、注意事项）

### 行装清单

- 5大类 27 项物品（防雨装备 / 登山装备 / 衣物 / 药品 / 其他）
- 必备物品标识
- localStorage 持久化勾选状态

### 交互与导航

- 顶部粘性导航栏（按日期快速跳转，滚动自动高亮）
- 底部日期浮窗（前后切换、回顶部按钮，移动端显示）
- 右侧快捷导航（天气 / 行程 / 美食 / 伴手礼 / 门票 / 清单）
- 滚动进度条
- 横向滑动切换日期（移动端手势）
- 出发倒计时（出发前 / 旅途中 / 行程结束后不同状态展示）
- 今日自动定位（行程期间自动滚动到当天卡片）
- 卡片入场动画（IntersectionObserver）
- 触控反馈（移动端触摸态）
- 无障碍支持（键盘导航、ARIA 属性）

### 深色模式

- 手动切换 + 跟随系统偏好
- 浅色模式：水墨丹青配色（宣纸暖白 + 墨绿主色 + 赭石点缀）
- 深色模式：朱砂墨韵配色（浓墨黑底 + 烫金主色 + 朱砂亮红）
- SVG 背景装饰（山水纹样 / 墨韵纹样）
- 平滑过渡动画

## 技术架构

```
src/                        源代码目录（Cloudflare Pages 构建目录）
  index.html              主页面（HTML 结构）
  style.css               全部样式（CSS 变量主题系统 + 响应式）
  main.js                 入口逻辑
  js/                     JS 模块
    utils.js              工具函数
    theme.js              主题切换
    weather.js            天气获取
    cards.js              卡片渲染
    nav.js                导航逻辑
    loader.js             数据加载
  data/                   数据文件
    day12.js ~ day20.js   每日行程数据（9个文件）
    days.js               组装 dailyData 数组
    food.js               美食数据 + 景点导航映射 + 景点美食关联
    gifts.js              伴手礼数据
    checklist.js          行装清单数据
```

**技术栈**:

- 纯静态 HTML + CSS + JavaScript，无框架依赖
- CSS 变量实现主题切换（浅色 / 深色）
- Open-Meteo API（免费天气接口，无需 API Key）
- 高德地图 URI API（路线规划、地点导航）
- Font Awesome 6 图标
- Google Fonts（Inter + ZCOOL KuaiLe）
- localStorage 持久化（CheckList 勾选状态、主题偏好）

## 数据格式

### 每日行程 (day12.js ~ day20.js)

每个文件导出 `window._dayXX` 对象：

```javascript
{
    date: "6/12 周五",           // 显示日期
    dateKey: "2026-06-12",       // ISO 日期键（用于天气查询）
    dayTitle: "启程入山",         // 当天主题
    walkDistance: "0.5km",       // 步行距离
    totalDuration: "2h",         // 总时长
    intensity: 1,                // 强度（1=轻松, 2=适中, 3=充实）
    accommodation: "酒店名（地址，评分，特色）",
    blocks: [                    // 时间块数组
        {
            time: "✈️ 下午 14:30",          // 时间标签
            duration: "2h",                  // 游玩时长（可选）
            desc: "详细描述...",             // 行程描述（支持 HTML）
            meal: "午餐推荐...",             // 餐饮推荐（可选）
            mealDetail: { ... },             // 详细餐饮信息（可选）
            tips: [                          // 分类提示（可选）
                { category: "标题", icon: "fas fa-xxx", items: ["提示1", "提示2"] }
            ],
            transport: "交通信息...",         // 交通方式（可选）
            rainPlan: "雨天备选方案...",      // 雨天方案（可选）
            realReview: "真实评价...",        // 游客评价（可选）
            note: "注意事项...",             // 注意事项（可选）
            spots: ["景点1", "景点2"],       // 关联景点（用于链接生成）
            sunriseFor: "wuyi",             // 日出城市（用于动态更新时间）
            type: "meal"                    // 块类型（meal=餐饮块）
        }
    ]
}
```

### 美食数据 (food.js)

```javascript
{
    name: "岚谷熏鹅",
    region: "wuyi",              // wuyi / lushan / jiujiang
    location: "武夷山 · 三姑度假区老街",
    desc: "详细描述...",
    tags: ["非遗", "必吃"],
    type: "main",               // breakfast / main / snack / dessert
    rating: 5,                  // 评分（5星制）
    priceRange: "¥45-80/份",
    mapUrl: "https://uri.amap.com/..."   // 高德导航链接
}
```

### 伴手礼数据 (gifts.js)

```javascript
{
    name: "大红袍茶叶",
    city: "武夷山",
    region: "wuyi",
    desc: "详细描述...",
    price: "200-800元/斤",
    where: "购买地点",
    tags: [{ text: "必买", type: "hot" }]   // hot / local / tea
}
```

### 行装清单 (checklist.js)

```javascript
{
    name: "折叠伞",
    category: "防雨",           // 防雨 / 登山 / 衣物 / 药品 / 其他
    essential: true             // 是否必备
}
```

## 设计风格

**浅色模式 -- 水墨丹青**:

- 背景：宣纸暖白 `#f5f0e8`，渐变过渡
- 主色：墨绿 `#2d5a27`
- 点缀：赭石暖红 `#c75b39`
- SVG 山水纹样背景装饰

**深色模式 -- 朱砂墨韵**:

- 背景：浓墨黑 `#0A0A0F`
- 主色：烫金 `#D4A853`
- 点缀：朱砂亮红 `#E85D4A`
- SVG 墨韵纹样 + 径向光晕

**字体**:

- 正文：Inter（西文）+ 系统中文字体
- 标题：ZCOOL KuaiLe（站酷快乐体，手写风格）
