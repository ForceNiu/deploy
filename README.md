# ⛰️ 武夷山 · 庐山 · 九江 旅行计划

> 竹筏 · 茶香 · 云海 · 慢生活 | 6.12 – 6.20

## 📋 项目简介

这是一份精心规划的9天旅行计划页面，涵盖江西武夷山、庐山、九江三地的完整行程安排。

## ✨ 功能特性

### 🗓️ 行程规划
- 每日详细行程安排，包含具体出发时间
- 交通方式、耗时、费用信息
- 雨天备选方案

### 🌤️ 天气预报
- 实时天气数据（Open-Meteo API）
- 历史同期天气参考
- 天气预警提醒
- 三城市天气对比

### 🍽️ 美食推荐
- 15道地方特色美食
- 按城市筛选（武夷山/庐山/九江）
- 餐厅导航链接

### 🎁 伴手礼推荐
- 9款地方特产
- 按城市筛选
- 价格参考

### 📝 实用工具
- 行装清单（防雨装备/药品/其他）
- 门票价格汇总
- 景点导航链接
- 滚动进度条
- 快捷导航

## 🎨 设计风格

采用「水墨丹青」配色方案，古典雅致：
- 宣纸暖白背景
- 墨绿主色调
- 赭石暖红点缀

## 🚀 技术栈

- 纯静态 HTML + CSS + JavaScript
- 无框架依赖，单文件部署
- 响应式设计，支持移动端
- Font Awesome 图标
- Google Fonts 字体

## 📱 访问方式

- **在线访问**：https://nzmin.gitee.io/deploy
- **本地使用**：直接在浏览器中打开 `index.html`
